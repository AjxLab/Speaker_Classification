Speaker Classification
======================

Templates for classifying multiple audio types.


## Usage
### Build Teacher Data
```sh
$ python build.py
```
### Train
```sh
$ python train.py
```


## Requirements
- macOS
- Python 3.7.4


## Installation
```sh
$ pip install -r lib/requirements.txt
```
